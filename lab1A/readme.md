# IL2206 Embedded Systems — Lab 1A

This repository contains code and materials for **Lab 1A: Concurrent Software Development in Ada** in the KTH course **IL2206 Embedded Systems**.

The lab focuses on concurrent programming in Ada, including:  
- Using concurrency features such as `task`, `protected object`, and rendezvous  
- Implementing a simple **counting semaphore**  
- Solving the multi-producer / multi-consumer **Producer–Consumer** problem with proper synchronization and mutual exclusion  
- Using a bounded circular buffer for inter-task communication  

------

本仓库包含 KTH 课程 **IL2206 Embedded Systems** 中实验 **Lab 1A: Concurrent Software Development in Ada** 的代码与相关文件。

该实验通过 Ada 语言练习并发程序设计，主要包括：  
- 使用 `task`、`protected object`、rendezvous 等并发机制  
- 实现简单的 **计数信号量（semaphore）**  
- 在多生产者–多消费者（Producer–Consumer）模型中实现同步与互斥  
- 使用有界循环缓冲区（bounded buffer）在任务之间传递数据
